#  To-Do List
A simple and interactive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**. This project was created for the purpose of practicing JavaScript fundamentals, DOM manipulation, and event handling.

##  Features
- **Add tasks** by typing and pressing Enter
- **Delete tasks** using the trash icon
- **Update tasks** with the edit icon
- **Mark tasks as completed** using checkbox
-  All changes reflect in real-time (no page reload)

##  Purpose

This project was built to:
- Practice **JavaScript** skills in a real-world UI
- Learn how to dynamically manipulate the **DOM**
- Handle **events** like clicks, input, and keypresses
- Improve understanding of **HTML structure** and **CSS layouting**

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap Icons (for UI icons only)
